## Installation

Use the package manager [npm](https://www.npmjs.com/) to install application.

```bash
- git clone https://github.com/Lukaina/Test-Design-Frontend-Developer.git
- Open text editor (e.g. Visual Studio Code)
- npm install
- npm start
```

# List Tasks

- Install react-MDL
- Install react-icons
- Make template
